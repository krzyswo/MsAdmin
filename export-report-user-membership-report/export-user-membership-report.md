Name: export-user-membership-report
Description: This PowerShell script exports Office 365 groups a user is member of, allowing generation of detailed user membership reports with advanced filters.
Prerequisites: PowerShell, Office 365 administrative access
Usage: Execute the script with necessary parameters to generate the report.
Output: The script exports a CSV file containing the list of Office 365 groups a user is member of.
Notes: The output CSV format includes detailed group membership information. The original image and external links have been removed for privacy.
