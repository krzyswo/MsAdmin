---
Name: export-report-permissions-report
Description: This PowerShell script helps to get calendar permissions for all the Office 365 mailboxes and exports 6 different mailbox permission reports to CSV.
Prerequisites: Office 365 administrative credentials and PowerShell.
Usage: Execute the script with administrative privileges in PowerShell to generate the reports.
Output: The script exports an output CSV file.
Notes: The output CSV file format and data fields are designed to provide a comprehensive view of calendar permissions across all mailboxes.
